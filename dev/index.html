ComponentArrays.Axis — Type
ax = Axis(nt::NamedTuple)

Axes for named component access of CArrays. These are a little confusing and poorly thought-out, so maybe don't use them directly.

Examples

julia> using ComponentArrays

julia> ax = Axis(a=1, b=2:3, c=(4:10, (a=(1:3, (a=1, b=2:3)), b=4:7)))
Axis{(a = 1, b = 2:3, c = (4:10, (a = (1:3, (a = 1, b = 2:3)), b = 4:7)))}()

julia> A = [100, 4, 1.3, 1, 1, 4.4, 0.4, 2, 1, 45];

julia> cvec = CArray(A, ax);

julia> cmat = CArray(A .* A', ax, ax);

julia> cmat[:c,:c] * cvec.c
7-element Array{Float64,1}:
  2051.52
  2051.52
  9026.688000000002
   820.608
  4103.04
  2051.52
 92318.4

ComponentArrays.CArray — Type
x = CArray(nt::NamedTuple)
x = CArray{T}(nt::NamedTuple) where {T}

Array type that can be accessed like an arbitrary nested mutable struct.

Examples

julia> using ComponentArrays

julia> x = CArray(a=1, b=[2, 1, 4], c=(a=2, b=[1, 2]))
CArray{Float64}(a = 1.0, b = [2.0, 1.0, 4.0], c = (a = 2.0, b = [1.0, 2.0]))

julia> x.c.a = 400; x
CArray{Float64}(a = 1.0, b = [2.0, 1.0, 4.0], c = (a = 400.0, b = [1.0, 2.0]))

julia> x[5]
400.0

julia> collect(x)
7-element Array{Float64,1}:
   1.0
   2.0
   1.0
   4.0
 400.0
   1.0
   2.0

ComponentArrays.getaxes — Method
getaxes(x::CArray)

Access .axes field of a CArray. This is different than axes(x::CArray), which returns the axes of the contained array.

Examples

julia> using ComponentArrays

julia> ax = Axis(a=1:3, b=(4:6, (a=1, b=2:3)))
Axis{(a = 1:3, b = (4:6, (a = 1, b = 2:3)))}()

julia> A = zeros(6,6);

julia> ca = CArray(A, (ax, ax))
